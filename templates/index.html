Welcome! This site is an example of what you can build with Python. Libraries such as Flask, sqlalchemy, requests, arcgis were used to build the backend. Gunicorn is used to serve this on Heroku.

Resources in this site:

  • Hello. Called as /hello?name='value'
  • Generate Unique Random numbers. Called as /genUniqueRandom?numRandom=15&upperLimit=25
  • Landsat image of an address
  • File upload
  • Geocode address. Called as /geocodeAddress?address='1600 Pennsylvania ave, Washington DC'
  • Get all addresses from DB
  • Get address in id 2
